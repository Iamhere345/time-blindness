# Time Blindness
This is a game about time management. School starts tomorrow and you have left everything to the last minute. To Get everything done, you must complete minigames effectively and manage your distractions. Supports Windows, Linux, MacOS (Universal), and web.

# Controls
Use the mouse to click on either a todo list entry or the thought bubble idea to start a minigame. Each minigame will show the controls before it begins (involves either the mouse or keyboard). Complete each item on the todo list before time runs out to win. Theres more info on how to play on the itch.io page.
